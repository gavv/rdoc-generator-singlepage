# RDoc::Generator::RSinglePage

Single page HTML5 generator for Ruby RDoc.

*Work in progress!*

## Features

* Generate single standalone HTML5 page suitable for offline use.
* Themes support.
* Exclude classes or members by regex.
* Group class members by regex.
